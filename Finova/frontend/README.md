# Finova Frontend

Interface React avec Tailwind CSS et Shadcn/UI.

## Structure

- `src/components/` - Composants réutilisables
- `src/pages/` - Pages principales
- `src/hooks/` - Custom hooks (API, UI)
- `src/App.jsx` - Composant racine
- `src/index.css` - Styles Tailwind

## Développement Local

```bash
# Installer les dépendances
npm install

# Lancer le développement
npm start

# Build production
npm run build
```

## Fonctionnalités

- Dark Mode épuré
- Upload drag-and-drop
- Graphiques en temps réel (Recharts)
- Responsive design
- Animation fluides

## Variables d'Environnement

- `REACT_APP_API_URL` - URL du backend (par défaut: http://localhost:8000)
