# Finova Project

## Structure du projet

```
.
├── backend/          # API Backend (Python/FastAPI)
├── frontend/         # Interface Utilisateur (React)
├── docs/             # Documentation du projet
├── data/             # Données sensibles (non versionnées)
├── docker-compose.yml
├── setup.sh
└── Taskfile.yml
```

## Organisation

- **backend/** : Code du serveur backend
- **frontend/** : Code de l'interface utilisateur
- **docs/** : Documentation complète du projet
- **data/** : Données bancaires et fichiers sensibles (ignorés par git)

## Configuration

Consultez `docker-compose.yml` pour la configuration Docker.

Pour plus de détails, voir les fichiers dans `docs/`.
